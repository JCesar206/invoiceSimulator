# invoiceSimulator
Simulador Simple de Facturas
Simulador simple de facturas el cual permite agregar nuevos productos, eliminarlos y va sumando el total y añade el IVA.
Una vez que se termina de agregar la información permite limpiar los campos o descargar un archivo PDF de la información ingresada para imprimirla.
Cuenta con links a redes sociales.
